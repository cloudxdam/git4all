Acerca de Git4All

Git4All es una chuleta interactiva para consultar comandos Git rápidamente.

Pulsa un botón y la aplicación mostrará el comando, una breve explicación y un aviso si conviene usarlo con cuidado.

Uso

  • Busca la categoría que te interesa.
  • Pulsa el botón del comando que quieras consultar.
  • Copia el comando generado.
  • Sustituye los marcadores como <archivo>, <rama>, <url> o <hash>.
